# Changelog

All notable changes to this project are documented in this file.

## [1.0.0] - 2026-06-21

### Added

- Multi-module architecture (`app`, `feature:home`, `core:*`)
- Jetpack Compose + Material 3 design system module
- Hilt dependency injection across layers
- Domain use cases with JVM unit tests
- Home feature ViewModel with `StateFlow` UI state
- Detekt static analysis configuration
- GitHub Actions CI (Detekt, tests, debug APK)
- Macrobenchmark module with startup benchmarks and baseline profile generator
- MIT license and contributing guide
